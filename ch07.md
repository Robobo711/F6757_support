## 7-4 Channel 讓你的 Telegram 住著 24 小時待命的工程師

### 設定 Claude Code CLI 的 Telegram 外掛

**step 01：** 建立 Telegram Bot

請參考 6-3 節來設定新的 Telegram Bot（記得要複製新的 Bot 的 token）。

**step 02：** 在 Claude Code 安裝 Telegram 外掛程式

```md
/plugin install telegram@claude-plugins-official
```

```md
/reload-plugins
```

**step 03：** 設定 Telegram 外掛

```md
/telegram:configure <你的 token>
```

**step 04：** 安裝 Bun

- Windows：

```md
powershell -c "irm bun.sh/install.ps1 | iex"
```

- macOS / Linux：

```md
curl -fsSL https://bun.sh/install | bash
```

**step 05：** 啟動 Telegram 外掛程式

```md
claude --channels plugin:telegram@claude-plugins-official
```

**step 06：** 配對 Telegram 外掛與 Telegram Bot

```md
/telegram:access pair <你的配對碼>
```

**step 07：** 鎖定存取權

```md
/telegram:access policy allowlist
```
